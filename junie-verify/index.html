A DESIGN EXPLORATION FOR JUNIPER SQUARE

The close is drafted by an agent.
Who signs it?

Juniper Square's bet is that agents will run fund operations with an expert in the loop. This prototype explores the moment that bet becomes an interface: a fund controller reviewing Junie's draft quarter-end close for a real estate fund — verifying, correcting, and teaching the agent — before capital account statements reach 52 LPs.

◆Confidence is a first-class data type — every figure carries a verification state, not just a value.
◆Every number can answer "where did you come from?" — page-level provenance, one click away.
◆Corrections are training signal — expert review compounds; it isn't a compliance checkbox.
◆Autonomy is earned, not configured — oversight loosens the way it would for a new hire.

Design system notes

What this prototype is made of — built to extend an existing system, not fork it.

Principles for agentic-AI surfaces

Four rules this concept encodes. They generalize to any surface where an agent's output needs an expert's sign-off — quarter-end close, DDQ responses, KYC checks, distribution notices.
01
Confidence is a first-class data type
Every figure carries a verification state (verified / needs review / discrepancy) with a numeric confidence — rendered consistently everywhere the figure appears.
02
Every number answers "where did you come from?"
A calculation trace plus page-level source citations, one click from any value. Trust is inspected, not asserted.
03
Corrections are training signal
Approve / correct / return aren't just workflow states — the UI shows what the agent learned, so review effort visibly compounds.
04
Autonomy is earned, not configured
Oversight levels unlock with track record, like a new hire earning trust. The control communicates *why* it's locked, not just that it is.

Semantic status tokens

Status colors carry meaning and are never used decoratively. One hue per verification state, shared across chips, row tints, banners and meters — so "amber" means the same thing on every screen of the suite.
--status-verifiedmodels agree · auto-cleared
--status-reviewbelow confidence threshold
--status-discrepancyvalidators disagree
--actor-agentJunie's voice & actions
--actor-validatorindependent recomputation
--actor-experthuman sign-off · primary

Financial numerals

All figures use tabular lining numerals (font-variant-numeric: tabular-nums) so columns scan vertically, with accounting-style parentheses for outflows. Small detail, disproportionate trust payoff.
